# Запуск

## Весь бэкенд одной командой

```bash
cd deploy
docker compose up --build
```

Поднимает: PostgreSQL (5432), Redis (6379), ClickHouse (8123),
OTel Collector (4317/4318), Jaeger UI (16686), quotes (8081),
data-service (8082), gateway (8080). Котировки идут от встроенного
GBM-симулятора (`QUOTES_SIMULATE=1`).

Проверка:

```bash
curl localhost:8080/health
curl -X POST localhost:8080/auth/register -H 'Content-Type: application/json' \
     -d '{"login":"demo","password":"demo123"}'
docker exec trading-redis redis-cli SUBSCRIBE quotes:ticks
```

## С реальным драйвером (только Linux)

```bash
cd stock_gen_driver && make && sudo insmod stock_gen.ko
# у quotes убрать QUOTES_SIMULATE и пробросить устройства в контейнер,
# либо запустить quotes-go на хосте: go run . -addr :8081 -redis localhost:6379
```

## Сервисы по отдельности (разработка)

```bash
# инфраструктура
docker compose up postgres redis clickhouse

cd quotes-go         && go run . -simulate -redis localhost:6379 \
                        -clickhouse http://trading:trading@localhost:8123/trading
cd data-service-ktor && ./gradlew run
cd gateway-ktor      && ./gradlew run
```

## Android

```bash
cd android-app && ./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

Эмулятор ходит на хост через `10.0.2.2:8080` (зашито в `BuildConfig.GATEWAY_URL`).

## Нагрузочный тест

```bash
ulimit -n 65535
cd load-simulator
go run . -clients 10000 -ws-clients 1000 -ramp-up 60s -duration 90s -interval 5s
# gateway запускать с RATE_LIMIT_RPS=200000 BCRYPT_COST=6 (всё с одного IP)
```
