# Торговая платформа — обзор

Учебный проект: экосистема биржевого терминала от драйвера ядра Linux до
Android-приложения, рассчитанная на **10 000 одновременных клиентов**.

## Компоненты

| # | Компонент | Стек | Папка |
|---|-----------|------|-------|
| 1 | Android-приложение | Kotlin, Jetpack Compose | `android-app/` |
| 3 | Gateway | Kotlin, Ktor | `gateway-ktor/` |
| 4 | Data-сервис | Kotlin, Ktor, голый SQL | `data-service-ktor/` |
| 5 | Нагрузочный имитатор | Go | `load-simulator/` |
| 6 | Сервис котировок | Go | `quotes-go/` |
| 7 | Драйвер-имитатор | C (модуль ядра) | `stock_gen_driver/` |

Связующая инфраструктура: PostgreSQL, Redis, ClickHouse, OpenTelemetry + Jaeger
(`deploy/docker-compose.yml`).
