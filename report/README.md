# Отчёт

Отчёт по проекту в формате, приближенном к ГОСТ 7.32 (Typst).

```bash
typst compile main.typ main.pdf   # https://typst.app, бинарник в ~/.local/bin/typst
```

- `gost.typ` — оформление (поля 30/15/20/20, 14pt, нумерация, оглавление);
- `main.typ` — содержимое по структуре ТЗ: введение (актуальность, цель,
  задачи), 1. техническое задание, 2. архитектура системы, 3. описание
  реализации, 4. тестирование (включая результаты нагрузочного теста на 10k),
  заключение, список литературы, приложения (формат драйвера, схема БД,
  API gateway, протокол нагрузочного теста);
- `img/` — скриншоты Android-приложения (генерируются скриншот-тестами:
  `cd ../android-app && ./gradlew recordRoborazziDebug`, PNG в `app/screenshots/`);
- `load/load-test-10k.json` — сырой отчёт нагрузочного теста.
