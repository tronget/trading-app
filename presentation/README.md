# Презентация

Защитная презентация по проекту (Typst, формат 16:9, ~7 минут, 16 слайдов).
Структура повторяет отчёт: введение → ТЗ → архитектура → реализация →
тестирование → заключение.

```bash
typst compile slides.typ slides.pdf   # бинарник в ~/.local/bin/typst
```

- `slides.typ` — самодостаточный шаблон без внешних пакетов (компилируется офлайн);
- `img/` — скриншоты Android-приложения (копия из `../report/img/`).
